# wx-tree
 小程序树形组件 下拉框
# 功能
1.支持树形现实 仅一列 \
2.支持单选和多选 \
3.支持下拉框显示
# 借鉴
 [原项目](https://github.com/Kittyelf/wx_treeSelect) （若作者介意可删此项目） 很多代码百度CV 缝缝补补 封装的也比较烂 不过能跑就行
